# SCARD


## How to start
* run executable using 
  ```
  escript santorini_card
  ```
  (works on CADE MACHINE) 
* if press "\n" after running, it will use default card set
  ```
  [{"card":"Artemis"},{"card":"Prometheus"}] 
  ```

## Installation

```elixir
def deps do
  [
    {:santorini_card, "~> 0.1.0"}
  ]
end
```
